Para ejecutar el proyecto:

1. Ejecute el documento proyecto_integrador.sql desde workbench
2. Con VSC abra las carpetas 'DasBorad' y 'Servidor'
3. Ejecute npm install desde la terminal de ambos proyectos en VSC
4. En Servidor vaya a src/database/config.js en database escriba "proyecto_integrador" y configure el password según su configuracion local.
5. Ejecute npm run start desde la terminal de ambos proyectos en VSC
6. Valide la información pedida en el Sprint 8 en los link  http://localhost:3030/api/products o http://localhost:3030/api/users y para especificaciones del producto en http://localhost:3030/api/products/:id y http://localhost:3030/api/user/:id  
con :id menor  a 28 en productos y menor a 5 es usuarios.
7. Valide lo pedido en el Sprint 8 para el dasborad en el link http://localhost:3000/ desde alli puede usar los botones del menu lateral para navegar por las 2 vistas elaboradas.
